This page demonstrate gapi.client.rpcRequest(method,name,rpcParams) and gapi.client.RpcRequest object

Creates an RPC Request directly. The method name and version identify the method to be executed and the RPC params are provided upon RPC creation.

It return gapi.client.RpcRequest, an object encapsulating an RPC request. gapi.client.RpcRequest is not instantiated directly, rather it is returned by gapi.client.rpcRequest.

gapi.client.RpcRequest defines one method: gapi.client.RpcRequest.execute(callback)

Executes the request and runs the supplied callback on response.

See console for result.

Note: In order for the Oauth to work, pop-up block must be off.

If the authentication pop-up is delay (i.e. pop-up detected, user click on the link manually), the authentication would not work!
