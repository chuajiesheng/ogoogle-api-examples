This page demonstrate gapi.client.request(args) and gapi.client.HttpRequest object

Creates a HTTP request for making RESTful requests.

It return gapi.client.HttpRequest, an object encapsulating an HTTP request. gapi.client.HttpRequest is not instantiated directly, rather it is returned by gapi.client.request.

gapi.client.HttpRequest defines one method: gapi.client.HttpRequest.execute(callback)

Executes the request and runs the supplied callback on response.

See console for result.

Note: In order for the Oauth to work, pop-up block must be off.

If the authentication pop-up is delay (i.e. pop-up detected, user click on the link manually), the authentication would not work!
