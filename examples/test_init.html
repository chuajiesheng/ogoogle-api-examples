This page demonstrate gapi.auth.init(callback)

Initializes the authorization feature. Call this when the client loads to prevent popup blockers from blocking the auth window on gapi.auth.authorize calls.

See console for result.

Note: In order for the Oauth to work, pop-up block must be off.

If the authentication pop-up is delay (i.e. pop-up detected, user click on the link manually), the authentication would not work!
